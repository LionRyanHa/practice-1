# 한걸음 프로젝트 기록

마지막 갱신: 2026-06-19

## 사용자 우선 요청

- “테스트 사이트” 또는 “저번에 만든 것처럼 테스트”라고 하면 일반 로그인 화면이 아니라 **칭호 상점 전용 미리보기**를 연다.
- 전용 주소: `http://127.0.0.1:8000/?preview=shop`
- 이 주소는 로그인을 건너뛰고 칭호 부분부터 바로 보여야 한다.
- 로그인/회원가입/Supabase 인증 테스트는 사용자가 명시적으로 요청할 때만 일반 루트 주소를 사용한다.

## 앱 개요

- 프로젝트명: 한걸음
- 형태: 모바일 중심 한자 학습 웹 앱
- 주요 파일: `index.html`, `style.css`, `app.js`
- 백엔드: Supabase 인증, 프로필, 초대코드, 단원 구매, 퀴즈 제출, 순위표
- 화면: 로그인, 초대코드, 홈, 설정, 칭호 상점·보관함, 순위표, 학습 방식 선택, 단어 보기, 퀴즈, 결과

## 현재 구현된 주요 기능

- 이메일 로그인과 회원가입
- 초대코드 승인 및 관리자용 현재 코드 확인
- 5개 한자 단원과 포인트 기반 잠금 해제
- 객관식·직접 입력 방식의 한자 퀴즈
- 출석 보상, 연속 정답 보너스, 포인트, 레벨업
- 프로필 이름과 라이트/다크 테마 설정
- 원격 순위표와 현재 사용자 강조
- 모바일 스와이프 및 직접 입력 후 다음 문제 이동
- 단어 크게 보기와 좌우 이동

## 칭호 상점

- 기본 프레임 외 8개 칭호가 있다.
- 순서: 빨강, 주황, 노랑, 초록, 파랑, 남색, 보라, 무지개
- 앞 칭호를 구매해야 다음 칭호를 구매할 수 있다.
- 칭호별 배경색·장식·애니메이션이 상점 카드, 프로필 아바타, 순위표에 반영된다.
- 상점과 보관함에서 구매, 장착, 해제가 가능하다.

## 칭호 전용 테스트 모드

`app.js`의 `isShopPreviewMode()`가 URL의 `preview=shop`을 확인한다. 활성화되면:

- Supabase 로그인 초기화를 실행하지 않는다.
- 모든 칭호를 순서대로 구매해 볼 수 있도록 충분한 테스트 포인트를 넣는다.
- 보유/장착 상태를 기본 프레임으로 초기화한다.
- 상점 화면을 즉시 표시한다.
- `data-shop-preview="true"`를 설정해 미리보기용 애니메이션 동작을 유지한다.

## 테스트 실행 순서

1. 저장소 루트에서 `python -m http.server 8000`을 실행한다.
2. HTTP 200 응답을 확인한다.
3. 반드시 `http://127.0.0.1:8000/?preview=shop`을 연다.
4. 첫 활성 화면이 `shop-screen`인지, 로그인 화면이 비활성인지 확인한다.
5. 빨강부터 순차 구매와 보관함 장착이 동작하는지 확인한다.

## 작업 시 주의사항

- 현재 작업 트리에 사용자의 미커밋 변경이 있을 수 있다. 관련 없는 변경은 되돌리지 않는다.
- 실제 Supabase 설정값이나 인증 정보는 출력하거나 기록하지 않는다.
- 테스트 편의를 위해 일반 인증 흐름을 약화시키지 말고, 전용 쿼리 모드만 사용한다.

## 최근 수정 기록

- 2026-06-19: 퀴즈 시작을 사용자당 시간당 10회로 막던 `start_quiz_attempt` 제한을 제거했다. 기존 Supabase 프로젝트용 `supabase-remove-quiz-rate-limit.sql` 패치를 추가하고, 시작 버튼 연속 클릭에 의한 중복 요청을 방지했다.
- 2026-06-19: 파랑·남색·무지개 칭호 애니메이션의 반복 경계 순간 이동, 과도한 밝기 변화, 필터 재합성으로 인한 깜빡임을 완화했다. 반복 이동은 왕복형으로 변경했고 남색 밝기 필터는 투명도·크기 변화로 교체했다.
- 2026-06-19: 파랑·남색 칭호를 보라색과 동일한 회전 포털·궤도 링 구조로 교체했다. 무지개 칭호는 상점 카드와 순위표 모두 고채도·고불투명도 색상으로 강화했다.
- 2026-06-19: 최종 칭호 품질을 파랑(소형 단일 링), 남색(중형 포털·링·룬), 보라(대형 포털·강화 링·이동 별), 무지개(이중 링·고정 별) 순으로 차등화했다. 무지개의 불투명도 0 키프레임과 사라지는 광택·별 마크업을 제거했다.
- 2026-06-19: 파랑~무지개 링 기반 효과를 전면 폐기하고 전용 레이어 시스템으로 재설계했다. 파랑은 해류·포말, 남색은 심해 수정, 보라는 마력 결정·한문 룬, 무지개는 프리즘 왕관·광선·코어를 사용한다. 고급 칭호 애니메이션은 opacity/filter를 움직이지 않고 transform 또는 alternate background-position만 사용한다.
- 2026-06-19: 칭호 미리보기에서 prefers-reduced-motion이 새 애니메이션을 0.01ms로 정지시키던 문제를 수정했다. 보라는 원형 포털·회전 궤도를 복원하고, 파랑 해류·남색 수정·무지개 왕관의 이동 폭을 강화했다.
- 2026-06-19: 칭호 설명에 맞춰 빨강·주황은 배경 레이어만 강화하고, 노랑·초록은 정적 별·햇살과 잎·덩굴 요소를 추가했다. 남색은 기존 수정 애니메이션에 추가 수정, 굴절광, 광점을 더해 소폭 보강했다.
- 2026-06-19: 초록 칭호를 전용 정적 보태니컬 레이어로 고급화했다. 곡선 덩굴, 잎 4장과 잎맥, 씨앗 2개, 꽃점, 아이콘·배지 광택을 추가했으며 설명에 맞춰 애니메이션은 사용하지 않았다.
