운영 문의, 오류 제보, 새 프레임 제안을 받을 수 있는 기준 페이지입니다

카메라 권한 문제, 렌더링 오류, 모바일 사용성, 새로운 촬영 스타일 제안은 아래 기준에 따라 검토합니다.

01

문의 가능 범위

  • 카메라 시작 실패, 촬영 버튼 동작 문제, 저장 실패 제보
  • 프레임 스타일, 필터, 타이포그래피에 대한 개선 제안
  • 페이지 표기 오류, 정책 문서 수정 요청
  • 모바일 접근성이나 브라우저 호환성 문제 제보

02

운영 채널

공개 저장소 이슈 페이지를 기본 문의 채널로 사용합니다. 현재 확인 가능한 채널은 GitHub Issues이며, 관련 코드와 문서 변경 이력은 저장소 메인 페이지에서 확인할 수 있습니다.

운영 채널이 바뀌면 이 페이지와 정책 문서를 함께 업데이트합니다.

03

제보 시 포함하면 좋은 정보

  • 문제가 발생한 페이지와 선택한 촬영 모드
  • 사용한 브라우저와 기기 종류
  • 기대한 결과와 실제 결과, 가능하면 화면 캡처
  • 재현 가능한 순서와 카메라 권한 상태

04

응답 기준

모든 문의에 대한 즉시 응답을 보장하지는 않으며, 서비스 오류와 보안 이슈를 우선 검토합니다. 스타일 제안이나 기능 확장 요청은 운영 우선순위와 구현 범위에 따라 반영 여부가 달라질 수 있습니다.
