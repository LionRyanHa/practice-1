운영 문의, 오류 제보, 정책 관련 요청을 받을 수 있는 기준 페이지입니다

기능 이상, 계산 결과 표시 문제, 문서 수정 요청, 광고 정책 관련 문의는 아래 채널과 기준에 따라 검토합니다.

문의 가능 범위

  • 도구 오작동 또는 계산 오류 제보
  • 페이지 표기 오류, 오타, 설명 보강 요청
  • 광고 노출, 정책 문서, 개인정보 처리방침 관련 문의
  • 모바일 사용성이나 접근성 문제 제보

운영 채널

공개 저장소 이슈 페이지를 기본 문의 채널로 사용합니다. 현재 확인 가능한 채널은 GitHub Issues이며, 관련 코드와 문서 변경 이력은 저장소 메인 페이지에서 확인할 수 있습니다.

문의 채널이 바뀌면 이 페이지와 정책 문서를 함께 업데이트합니다.

제보 시 포함하면 좋은 정보

  • 문제가 발생한 페이지와 사용한 기능 이름
  • 입력한 값과 기대한 결과, 실제 결과
  • 사용한 브라우저와 기기 종류
  • 재현 가능한 순서 또는 화면 캡처

응답 기준

모든 문의에 대한 즉시 응답을 보장하지는 않으며, 서비스 오류와 정책상 중요한 이슈를 우선 검토합니다. 단순 제안이나 기능 확장 요청은 운영 우선순위에 따라 반영 여부가 달라질 수 있습니다.
