AI Face ID 프로젝트 소개

인공지능이 보는 시각적 특징을 분석하여 인물을 판별하는 기술 프로젝트입니다.

1. 프로젝트 개요

AI Face ID는 최신 웹 기반 딥러닝 기술을 활용하여, 사용자가 업로드한 사진 속 인물이 '원동균'님과 '이규진'님 중 누구와 더 닮았는지 분석해주는 AI 판별 서비스입니다. 별도의 프로그램 설치 없이 웹 브라우저에서 즉시 실행되며, 높은 접근성과 재미를 동시에 제공하는 것을 목표로 합니다.

2. 핵심 기술 (Technology Stack)

Google Teachable Machine

구글의 머신러닝 모델 학습 도구를 사용하여 수천 장의 인물 이미지를 학습시켰습니다. 전이 학습(Transfer Learning) 기법을 통해 적은 데이터로도 높은 정확도를 확보했습니다.

TensorFlow.js

학습된 모델을 웹 브라우저에서 직접 구동하기 위해 TensorFlow.js 라이브러리를 사용했습니다. 이를 통해 서버 비용을 절감하고 사용자의 프라이버시를 보호합니다.

MobileNet

이미지 분류를 위한 경량화된 CNN 모델인 MobileNet을 백본(Backbone)으로 사용하여, 모바일 기기에서도 빠르고 효율적인 추론이 가능합니다.

3. 개발 과정 및 데이터셋

정확한 판별을 위해 각 인물의 다양한 각도, 조명, 표정이 담긴 사진 데이터를 수집했습니다. 데이터 전처리 과정을 통해 얼굴 영역을 크롭(Crop)하고 정규화(Normalization)하여 모델 학습의 효율을 높였습니다.

  • Class 1 (원동균): 정면, 측면, 웃는 표정 등 데이터셋 500+장
  • Class 2 (이규진): 안경 착용 유무, 다양한 조명 환경 데이터셋 500+장

4. 향후 로드맵

현재는 특정 두 인물의 판별에 집중하고 있지만, 향후에는 사용자 피드백을 반영하여 다음과 같은 기능을 추가할 예정입니다.

  • ✅ 더 많은 인물 클래스 추가 및 다중 분류 지원
  • ✅ 실시간 웹캠 분석 기능 도입
  • ✅ 모바일 앱(Android/iOS) 출시
