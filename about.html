AI Face ID Demo 프로젝트 소개

웹 브라우저에서 실행되는 이미지 분류 모델을 설명하기 위해 만든 데모형 프로젝트입니다.

1. 프로젝트 목적

이 사이트는 얼굴 이미지를 기계학습 모델이 어떻게 분류하는지 보여주는 교육용 데모입니다. 실제 사람의 신원을 확인하거나 사실 여부를 판정하는 서비스를 제공하지 않으며, 결과는 참고용 시각화에 가깝습니다.

2. 기술 구성

Google Teachable Machine

예시 분류 모델을 제작하고 브라우저에서 쉽게 배포할 수 있도록 활용한 학습 도구입니다.

TensorFlow.js

학습된 모델을 사용자의 브라우저 안에서 실행해 서버 업로드 없이 추론 결과를 보여줍니다.

정적 웹 구조

HTML, CSS, JavaScript 중심의 정적 페이지로 구성해 이용 가이드와 정책 문서를 함께 제공합니다.

3. 데이터와 한계

이 데모는 사전 동의를 받은 예시 인물 2명을 기준으로 학습된 제한적 모델을 사용합니다. 따라서 다른 사람의 사진, 극단적인 조명, 낮은 해상도 이미지, 가려진 얼굴에서는 의미 있는 결과가 나오지 않을 수 있습니다.

  • 모델 출력은 제한된 예시 데이터에 대한 상대적 분류 점수입니다.
  • 정확도 수치를 보장하지 않으며 환경에 따라 결과 편차가 큽니다.
  • 민감한 의사결정에 사용할 수 없도록 명시적으로 제한합니다.

4. 운영 원칙

  • 동의 없는 타인 사진 업로드 금지
  • 결과를 오락 및 학습 목적 범위에서만 활용
  • 정책, 문의, 권리 침해 대응 정보를 사이트 내에 명시
