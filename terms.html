이용약관

최종 업데이트: 2026년 4월 30일

1. 서비스 성격

본 사이트는 브라우저 기반 포토부스와 프레임 합성 기능을 제공하는 웹서비스입니다. 사용자는 별도 계정 없이 카메라 또는 업로드 이미지를 바탕으로 결과를 생성할 수 있으며, 서비스는 일반적인 창작 편의 제공을 목적으로 합니다.

2. 이용자의 확인 책임

사용자는 촬영한 이미지에 대한 초상권, 저작권, 공개 범위를 스스로 확인해야 합니다. 운영자는 웹 도구를 제공하지만, 사용자가 생성한 이미지의 게시 또는 상업적 이용 적합성을 개별적으로 보증하지 않습니다.

3. 서비스 변경 및 중단

운영자는 기능 개선, 오류 수정, 정책 변경, 호환성 조정을 위해 사이트 일부를 수정하거나 중단할 수 있습니다. 중요한 변경이 있는 경우 관련 페이지에 업데이트 내용을 반영합니다.

4. 외부 서비스와 환경 제약

카메라 권한 허용, 브라우저 보안 정책, 기기 성능, 운영체제 제한에 따라 기능 동작 방식이 달라질 수 있습니다. 외부 링크나 리소스 이용 시에는 해당 서비스의 정책과 책임 범위가 함께 적용됩니다.

5. 금지 행위

  • 기능 오작동을 유도하기 위한 반복적인 비정상 입력 또는 자동화 요청
  • 타인의 권리를 침해하는 이미지 사용 또는 서비스 사칭
  • 보안 우회를 시도하거나 운영을 방해하는 행위

6. 문의

운영 관련 문의는 문의 페이지의 안내를 따릅니다. 정책과 운영 채널이 변경되면 해당 문서와 함께 갱신합니다.
