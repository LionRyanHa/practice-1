이 사이트의 목적

AI Face ID Demo는 웹 브라우저에서 작동하는 이미지 분류 모델의 동작 방식을 설명하기 위해 제작한 실험형 프로젝트입니다. 본 결과는 재미와 학습을 위한 참고 정보이며, 실명 확인이나 본인 인증 도구로 사용할 수 없습니다.

  • 분석 대상은 사전 동의를 받은 예시 인물 2명입니다.
  • 업로드 이미지는 브라우저 내부에서만 처리되며 서버에 저장하지 않습니다.
  • 민감한 얼굴 정보의 자동 식별, 감시, 채용 판단, 신원 확인 용도로 사용해서는 안 됩니다.

결과를 읽는 방법

데이터 처리 방식

업로드 이미지는 서버에 저장하지 않으며 사용자의 브라우저에서만 분석합니다. 다만 애드센스, 폰트 등 외부 서비스가 페이지 로드 시 일반적인 접속 정보를 처리할 수 있으므로 세부 내용은 개인정보처리방침에서 확인할 수 있습니다.

왜 광고 심사에서 문제가 될 수 있었나

광고 심사에서는 얇은 도구형 페이지, 과장된 성능 표현, 운영 주체 정보가 부족한 사이트, 실존 인물 얼굴을 가볍게 다루는 페이지를 불리하게 볼 수 있습니다. 그래서 이 사이트는 목적, 한계, 동의 요건, 데이터 처리 방식을 더 명확하게 설명하도록 개편했습니다.

현재 구성은 기능만 보여주는 한 페이지형 도구에서 벗어나 프로젝트 설명, 이용 가이드, 책임 있는 사용 원칙, 문의 경로, 정책 문서를 함께 제공하는 형태를 목표로 합니다.
